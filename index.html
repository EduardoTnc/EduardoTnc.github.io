Etiquetas semánticas y estructurales

div - Etiqueta contenedora que no tiene semámtica

header - Cabecera de un sitio o una sección

main - Define la sección principal del documentos, solo puede existir uno por documento

footer - Pie de página de un sitio web o de una sección

nav - Representa una navegación

article - Representa una sección autocontenida (que por sí sola se explica)

aside - Representa contenido complementario o secundario

section - Representa una sección de contenido genérico

address - Representa una información de contacto

Formularios

Elementos de Formulario

El elemento más básico son los input con su atributo type:

  • type="text"


  • type="checkbox"


  • type="radio"


  • type="date"


  • type="color"


  • type="button" value="Botón"


  • type="submit" (cuando está dentro de un formulario envía sus datos)


  • type="reset" (limpia los valores introducidos en el formulario)


  • type="file"


  • type="file" multiple


  • type="hidden" name="idioma" value="es" (no se ven, sirven para que cuando el formulario se envie tu como programador puedas enviar algún valor)


  • type="email"


  • type="number"


  • type="tel"


  • type="password"


  • type="search"


Siguen los textarea y select :

  • textarea name="" id="" cols="50" rows="8"
